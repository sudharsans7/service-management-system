# Service Request Management System

A web-based application that allows users to raise, track, and manage service requests in real time.

## Problem Statement
Manual service issue tracking through calls or messages leads to missed requests and lack of visibility. This application provides a centralized and structured solution.

## Features
- Raise service requests
- View all requests with status
- Real-time frontend-backend integration
- Clean and responsive UI

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express
- Tools: Git, GitHub

## How to Run
1. Navigate to backend folder and start server:
2. Open `index.html` in browser.

- Full-stack fundamentals
